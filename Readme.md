# AI Career Guidance Expert System

This project is a Rule-Based Expert System that recommends career paths based on user skills and interests.

## Features

- Rule Based AI
- Forward Chaining
- Multi-Step Inference
- Career Recommendations
- Streamlit UI

## Technologies

- Python
- Streamlit

## How it Works

The system accepts user interests and skills as facts.

Using Forward Chaining, predefined IF-THEN rules are applied to infer suitable career recommendations.

## Run

streamlit run app.py
